# 「纸间」笔记应用设计

## 产品方向

这是一个无需安装、打开即用的本地笔记应用。产品吸收成熟笔记软件最稳定的共同模式：Notion 的侧栏与快速检索、Bear 的标签和低干扰写作、Apple Notes 的置顶与富文本、Craft 的内容层次。视觉方向是“暖纸张 + 编辑部工作台”，避免常见的办公 SaaS 模板感。

## 功能范围

- 左侧：全文搜索、全部/置顶筛选、标签筛选、按更新时间排列的笔记卡片。
- 右侧：标题、标签、富文本正文、格式工具栏、字数与自动保存状态。
- 快捷操作：新建、置顶、删除并撤销、明暗主题、JSON 导入与导出。
- 数据：使用 `localStorage` 离线保存；首次打开提供三篇示例笔记。
- 键盘流：`Ctrl/Cmd + N` 新建，`Ctrl/Cmd + K` 搜索，`Ctrl/Cmd + S` 立即保存，`Esc` 返回列表/关闭浮层。
- 响应式：桌面双栏；小屏在列表和编辑器之间切换。

## 数据与安全

每篇笔记包含 id、标题、HTML 正文、标签、创建时间、更新时间和置顶状态。粘贴统一按纯文本处理，导入内容经过白名单清洗，链接只允许 http、https 和 mailto 协议，避免把脚本保存进页面。

## 测试标准

1. 首次加载能看到示例数据，刷新后内容不丢失。
2. 新建、编辑标题和正文后自动保存，列表摘要与排序同步更新。
3. 搜索可命中标题、正文、标签；置顶和标签筛选结果正确。
4. 删除后可撤销；导出文件结构有效；导入数据可读取且不会执行脚本。
5. 明暗主题可切换并记忆；常用快捷键有效。
6. 桌面双栏无溢出；窄屏能在列表和编辑器之间正常往返。
